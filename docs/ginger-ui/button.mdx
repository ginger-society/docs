# Button

The <Button label='Button' primary /> component is a versatile UI element that can be used for various interactions within your application. Below you will find detailed information about its props and how to use it.

## Props

| Prop             | Type                              | Default   | Description                                                           |
|------------------|-----------------------------------|-----------|-----------------------------------------------------------------------|
| `primary`        | `boolean`                         | `false`   | Determines if the button is the principal call to action on the page. |
| `backgroundColor`| `string`                          | `null`    | Specifies the background color of the button.                         |
| `size`           | `'small' \| 'medium' \| 'large'`  | `'medium'`| Specifies the size of the button.                                     |
| `label`          | `string`                          | Required  | The text to be displayed inside the button.                           |
| `onClick`        | `() => void`                      | `null`    | Optional click handler function for button click events.              |

## Example Usage

### Primary Button

<Button
  label="Primary Button"
  primary
  size="large"
  backgroundColor="#007bff"
  onClick={() => alert('Primary Button Clicked')}
/>

```jsx
<Button
  label="Primary Button"
  primary
  size="large"
  backgroundColor="#007bff"
  onClick={() => alert('Primary Button Clicked')}
```
