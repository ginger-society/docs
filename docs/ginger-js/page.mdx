# Page

In a Python environment, you can fetch data, interact with the database, and pass the data to your page.

### Convention
The server logic is placed alongside index.jsx or layout.jsx within the same folder and is named index.py.

### Server Example

Path Example : `src/app/index.py`

```python
def index(request):
    return {"name":"GingerJs"}

```

Note: Returned data will be accessible through props in Page


### Client Example

Path Example : `src/app/index.jsx`

```jsx
import React, { useEffect, useState } from "react";

function Index(props) {
  return (
    <div>
      <h1>Hello from{props.name}<h1>
    </div>
  );
}

export default Index;

```
Note: "`Index`" you can replace this with more meanigful name but dont forget to export it as default


### Middleware

You can also process request


Path Example: `src/app/middleware.py`
```python
def middleware(request,abort):
    token = request.headers.get('X-Auth-Token')
    if token != 'secret-token':
      return abort(401,{"error":"No Auth"})
    return
```
