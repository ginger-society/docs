# API 

Server-side API endpoints

Path Example : `src/app/api/product/index.py`

```python
def index(request):
    data = {}
    for key,value in request.args.items():
        data[key] = value
    return {"query":data,"id":id}
```


Now you can access this endpoint from UI

For instance, if you want to fetch data on click, you can achieve this by: 

```jsx
import React, { useState } from "react";
import { useParams } from "react-router-dom";

function Index(props) {

    const [data,setData] = useState({})
    const [err,setErr] = useState(false)

    const fetchData = ()=>{
        fetch("/api/product").then(async res=>setDate(await res.json())).catch(err=>setErr(true))
    }


    return (
        <div>
            <pre>
                data: {JSON.stringify(data)}
            </pre>   
            {err && (
                <p>
                    Error: {err}
                </p>
            )} 
            <button onClick={fetchData}>Fetch Data</button>
        </div>
    );
}

```
